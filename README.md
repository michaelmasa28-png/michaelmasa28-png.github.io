# Masa Web Developer Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🎨 Modern and responsive design
- ⚡ Smooth animations and transitions
- 📱 Mobile-friendly layout
- 💬 Contact form functionality
- 🚀 Easy to customize

## Project Structure

```
├── index.html      # Main HTML file
├── styles.css      # Styling
├── script.js       # JavaScript interactions
└── README.md       # Documentation
```

## How to Use

1. Edit the HTML content in `index.html`
2. Customize colors and styles in `styles.css`
3. Modify interactions in `script.js`
4. Replace placeholder project cards with your actual projects
5. Update contact form to route to your email service (Formspree, Netlify, etc.)

## Customization Tips

### Change Colors
Update the gradient colors in `styles.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add Projects
Add more `.project-card` divs in the projects section with your own content.

### Enable Email Notifications
For the contact form, integrate with a service like:
- [Formspree](https://formspree.io/)
- [Netlify Forms](https://www.netlify.com/products/forms/)
- [EmailJS](https://www.emailjs.com/)
